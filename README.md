
## Debian

| Script | Purpose |
|---|---|
| `debian/setup.sh` | Basic Debian server setup with essential packages |
| `debian/vm.sh` | Debian VM setup with essential packages and QEMU guest agent |

## Usage

Run the Debian server setup:

```bash
curl -fsSL https://raw.githubusercontent.com/node-w/scripts/main/debian/setup.sh | sudo bash
```
Run the Debian VM setup:

```bash
curl -fsSL https://raw.githubusercontent.com/node-w/scripts/main/debian/vm.sh | sudo bash
```
